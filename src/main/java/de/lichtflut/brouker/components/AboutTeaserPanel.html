Einsatzgebiete und Möglichkeiten

Glasnost ist eine web basierte Plattform zur Verwaltung und Strukturierung von Informationen deines Unternehmens.

Wir haben es so entwickelt, dass man es sehr einfach anpassen und erweitern kann, z. B. für diese Zwecke:

  • Enterprise Architecture Management
  • Kunden- und Lieferantenverwaltung
  • (IT) Configuration Management
  • Task und Ticket System
  • Personalverwaltung - mit Skills und Expertisen
  • Wissensbilanzierung und -Management
  • ...
